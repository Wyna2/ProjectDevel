# ProjectDevel
'ProjectDevel'은 개발에 열정을 가진 사람들을 위한 공간을 만듬을 목적으로 한 Java Wep project 입니다.<br>
큰 카테고리는 그라운드(그룹), 커뮤니티, 중고장터, 챌린지 로 구성되어 있습니다.<br>
